Privacy Choices

Privacy Choices is a JavaScript library that allows you to give the users of your website choices regarding the use of cookies and other web storage technologies on their device. Configured correctly, this library should make your site compliant with EU legislation.

Examples // No categories

This example uses the default configuration with no categories defined. This is the minimal use case, where a site uses only 'necessary' cookies that it must inform the user about. There is no requirement to get the user's consent because there are no non-necessary categories in use. The library therefore just lets the user dismiss the notification. The settings menu remains so that the user can get more information, including viewing your privacy policy.

The only use case more minimal than this one would be when no cookies are used on a site whatsoever. In such a case there would be no need to use this library!

Having different configurations of consent choice categories on the same domain isn't a supported usage of this library. Therefore, if you encounter any issues with the library on this demo site please use the link provided on every demo to clear cookies for this domain prior to raising an issue on GitHub. This will reset the library as if you are visiting for the first time. Production usage of this library should use the same categories configuration across a domain so this would not be a problem for your users.
