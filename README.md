## Development
### Prerequisites
- [Node.js](https://nodejs.org/)

### Building
From a terminal:
```console 
$ npm install
$ npm run build
```

### Using
After building, reference `build/index.bundle.js`.

To start a live-updating webpack:
```console
$ npm run watch
```
